# Random Quote Generator

## Автор: Иван Иванов

## Описание
Приложение для генерации случайных цитат с возможностью фильтрации по автору и теме, сохранения истории в JSON и загрузки обратно.

## Использование
1. Запустите `quotes.py`.
2. Нажмите «Сгенерировать цитату».
3. Используйте фильтры по автору и теме.
4. История сохраняется в `quotes.json`.

## Примеры использования/тесты
- Генерация случайной цитаты.
- Фильтрация по автору: "Бенджамин Дизраэли".
- Фильтрация по теме: "мотивация".
- Проверка сохранения и загрузки истории.
